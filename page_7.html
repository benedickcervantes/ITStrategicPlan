IT Strategic Goals for 2026

Mitigate Critical Risks

Replace Critical Server UPS: Immediate replacement of failing power protection

Upgrade Critical Systems: Initiate RAM/SSD upgrades for critical users

Reinforce Network Infrastructure: Strengthen network reliability and connectivity

Email Migration: Migrate to Microsoft 365 with user training for Yondo CRM

Impact: Critical risk mitigation and immediate performance improvements

Strategic Modernization

Procure New Systems: Replace 14 outdated computer systems

Cloud Migration: Migrate physical server to cloud infrastructure

Modernize Communications: Implement Cloud PBX, SIP Trunking, and VoIP

Digital Presence: Develop website for condominiums

Impact: Long-term modernization and sustainability

Systematic Upgrades

Regularize Software Licenses: Acquire licenses for 16 unlicensed systems

Complete PC Upgrades: RAM & SSD upgrades for remaining systems

Upgrade Monitors: 18 units with existing monitors as second screens

Optimize Connectivity: Consolidate ISPs and optimize landline services

Implement IT Ticketing System: Deploy centralized support management for ticket tracking and performance metrics

Impact: Medium-priority items with significant user impact

These strategic goals will guide our 2026 IT investments, ensuring a more robust and productive environment
