Security and Compliance Risks

Unlicensed Software

16 systems are running unlicensed or cracked versions of Windows and Microsoft Office, representing 57% of the total audited systems (28-30 users).

Compliance Status: Non-compliant - Immediate action required in Phase 2

Associated Risks:

Legal penalties and compliance violations - Risk of fines and legal action from Microsoft
Increased vulnerability to cyberattacks - No security updates or patches available
System instability and crashes - Unlicensed software causing compatibility issues
No technical support - Unable to receive official Microsoft support for critical issues

Failing Server UPS

The APC UPS unit protecting the Fujitsu Primergy RX1330 M3 Rack Server is failing and requires IMMEDIATE replacement in Phase 1. Recommended: 2200 VA or 3000 VA capacity for critical network infrastructure protection.

Critical - Phase 1
Priority: Highest - Address immediately to prevent server failure and data loss

Associated Risks:

High risk of data loss during power outages - No protection for critical server
Extended system downtime affecting business continuity - Network services at risk
Compromised backup systems and data integrity - External storage at risk
Potential hardware damage from power surges or sudden shutdowns

Network Security Vulnerabilities

18 users affected by network infrastructure issues: unlabeled cables, shared ports (3-4 users per port), patch panel at capacity. Creates security blind spots and complicates incident response.

Priority: High - Address in Phase 2 for network security hardening

Associated Risks:

Security blind spots - Cannot trace network issues during security incidents
Shared network ports increase attack surface - One compromised device affects multiple users
Delayed incident response - Unlabeled cables slow down troubleshooting and isolation

CCTV Security Issues

HIKVISION DS-7616NI-Q2/16P NVR with 8 IP cameras - admin password unknown/not accessible. Cannot verify storage, enable cloud monitoring, or access security footage for incident investigation.

Priority: High - Critical security monitoring gap

Associated Risks:

No real-time threat detection - Cannot monitor security events as they occur
Cannot access footage for incident investigation - Evidence collection compromised
No cloud monitoring enabled - Cannot access remotely or receive alerts

Inadequate Backup Strategy

Single external HDD (4TB Seagate Backup Plus Hub) connected to server. No offsite backup, no cloud backup, no automated verification. Complete data loss risk in case of fire, theft, or hardware failure.

Priority: Critical - Address in Phase 2

Associated Risks:

Complete data loss in disaster scenarios - Fire, flood, or theft would destroy all backups
No backup verification - Cannot confirm backup integrity or successful restoration
Manual backup process - Risk of human error and missed backups

No Multi-Factor Authentication

Critical systems (server, network equipment, email) lack MFA protection. Single-factor authentication vulnerable to credential theft. M365 email rollout incomplete (35 users) - opportunity to implement MFA during migration.

Priority: High - Implement in Phase 1-2

Associated Risks:

Unauthorized access risk - Stolen or compromised passwords grant full system access
Server and network equipment vulnerable - Single password compromise = full network access
Email account takeover risk - Phishing attacks can compromise entire email system

Missing Security Policies

No documented password policies, antivirus management unclear, no endpoint protection strategy, no security awareness training program. Compliance gaps for data protection regulations including Data Privacy Act of 2012 (Philippines).

Priority: High - Address in Phase 2

Associated Risks:

Compliance violations - Risk of penalties under Data Privacy Act of 2012
Inconsistent security practices - No standardized approach to endpoint protection
Human error risk - No security awareness training increases phishing/social engineering vulnerability

No Incident Response Plan

No documented procedures for security breaches, data leaks, or cyberattacks. No defined roles, escalation paths, or recovery procedures. Extended downtime and data loss risk in case of security incident.

Priority: High - Address in Phase 2

Associated Risks:

Extended incident response time - No clear procedures delay containment and recovery
Unclear responsibilities - No defined roles lead to confusion during security incidents
Increased business impact - Delayed response increases financial and reputational damage

WiFi Security Gaps

2 WiFi access points (Ruijie Reyee RG-RAP2260) - no guest network separation, unclear access controls, potential unauthorized network access. 18 users experiencing network issues may indicate security misconfigurations.

Priority: Medium-High - Address in Phase 2

Associated Risks:

Unauthorized network access - Visitors/guests can access internal network resources
Weak or default WiFi passwords - Easy for attackers to gain network access
Network segmentation failure - No isolation between guest and corporate networks
